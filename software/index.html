Here’s some of my standalone software projects that I think are pretty cool.

Global illumination path tracer

GitHub Repo

A Monte Carlo global illumination path tracer written in C++, using the Intel Embree kernels for ray tracing. I implemented this path tracer while reading the excellent book Physically Based Rendering. This Cornell box (with the Stanford dragon, of course) demonstrating different physically-accurate materials took 30 minutes to render.

This row of spheres demonstrates rendering with depth of field (the aperture being ƒ/16 in this image). The rendering was stopped after about 20 minutes and 800 iterations. (All of these timings were recorded on a 2.3 GHz Intel Core i5.)
