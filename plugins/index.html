Plugins

This is a collection of plugins I’ve developed for Maya, 3ds Max, and Houdini. All of them are open-source, and some of them have pre-compiled binaries.

Mannequin for Maya

Download GitHub Repo

Mannequin is a plugin for Maya that allows you to animate an FK (forward kinematics) character rig by interacting directly with the character mesh. Simply hover over the mesh, and Mannequin will detect and highlight parts of the mesh corresponding to different bones in the rig.

In the above screenshot, the lower lip has an active translation manipulator. The pointer is hovered over the outer-left eyebrow, which highlights it in the viewport.

Built using C++, Python, and MEL. Requires Maya 2015 SP4 or later. Pre-compiled binaries available for Mac OS X and Windows.

iWave for 3ds Max

Download GitHub Repo

Based on Jerry Tessendorf’s ocean simulation course notes from Siggraph 2004. This 3ds Max plugin supports generating ambient waves on an open ocean and dynamic waves as a result of object collisions.

At the heart of the process, it uses an FFT to sum many individual waves constructed from a statistical model. It supports dynamic interaction with collision objects by using a convolution kernel over a grid of wave points (Tessendorf’s “iWave” method).

Stripped-down version for Maya

GitHub Repo

A stripped-down version of this plugin without the dynamics capabilities is available for Maya. The Maya version still includes the ambient wave functionality.

Terrain Generation Plugins

3ds Max Modifier Houdini SOP

In the video above, the desert scene outside the window is entirely procedurally-generated.

These plugins—a 3ds Max modifier and a Houdini SOP—implement the diamond-square algorithm. The algorithm can generate terrain of varying roughness, from deserts to mountains.

MotionBuilder + Windows Phone

Windows Phone App MoBu Plugin

Use your phone for motion capture! With this Windows Phone app and MoBu plugin, you can transmit your phone’s physical 3D orientation to MotionBuilder in realtime.

The TCP server and MoBu plugin run on a local computer to receive the phone’s orientation, letting you control objects in MotionBuilder by physically rotating your phone.

Check out the GitHub repos for more information on how the app, the TCP server, and the MotionBuilder plugin interact.
